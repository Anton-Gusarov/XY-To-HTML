On the left - absolute positioned elements, Right - HTML produced page. These elements are from data.js and processed through getRowsAndCols in order to get grid data structure necessary for HTML rendering. Also a valuable feature can be observed by calling doPushdown(60«or any number in px to grow») from dev console. Black components are growing and pushing below components according to the html rendering data structure. It happens in left side - absolute positioned. In doPushdown there is no any dependency on DOM.
